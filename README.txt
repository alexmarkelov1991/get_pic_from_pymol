Запуск контейнера:
в папке с проектом собрать контейнер командой:
docker-compose build
запустить командой:
docker-compose up

По запросу:
http://127.0.0.1:8080/
в ответе приходит список доступных молекул.
http://127.0.0.1:8080/picture/{название молекулы из списка}
в ответе приходит картинка выбранной молекулы.
